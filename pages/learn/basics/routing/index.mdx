So far, the One App app we created only has one page. In general websites and web applications could have many different pages.

Let's explore how to add routing to our application.

## What You’ll Learn in This Lesson

In this lesson, you will:

- Learn how to use [one-app-router](https://github.com/americanexpress/one-app-router).
- Learn how to route to other modules using [holocron-module-route](https://github.com/americanexpress/holocron/tree/master/packages/holocron-module-route).
- Learn how to route to components within your module.

> If you’re looking for detailed documentation on One App routing, take a look at the <a href="https://one-amex-docs.americanexpress.com/en-us/one-app/api/modules/routing" rel="noopener noreferrer" target="_blank">routing documentation</a>.
